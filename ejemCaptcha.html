¿Cómo poner CAPTCHA en HTML?

Para añadir un CAPTCHA a tu formulario HTML, puedes utilizar la opción de reCAPTCHA de Google.

Primero, debes registrar la URL de tu sitio web en Google reCAPTCHA y obtener las claves necesarias. Luego, debes copiar y pegar el siguiente código dentro del head de tu documento HTML para cargar la API de reCAPTCHA:

<script async src="https://www.google.com/recaptcha/api.js"></script>

Después, dentro de tu formulario, debes insertar el siguiente código:

<div id="recaptcha" class="g-recaptcha" data-sitekey="TU_CLAVE_DEL_SITIO"></div>

Recuerda reemplazar TU_CLAVE_DEL_SITIO con la clave del sitio que obtuviste de Google reCAPTCHA.
