Ejemplo de Solicitud AJAX con JavaScript

Para crear una solicitud AJAX completamente con JavaScript, sin necesidad de usar librerías adicionales, seguimos estos pasos:

1. **Crear el objeto XMLHttpRequest**: Este objeto será el encargado de realizar la solicitud asíncrona al servidor. Se utiliza para hacer peticiones HTTP y recibir respuestas sin necesidad de recargar la página.

2. **Configurar el evento de cambio de estado de la solicitud**: Cada vez que el estado de la solicitud cambie, debemos ejecutar una función para manejar la respuesta. Usamos el evento onreadystatechange para ello.

3. **Verificar el estado de la solicitud**: La solicitud estará lista cuando readyState sea 4 (completada) y status sea 200 (respuesta exitosa del servidor). Si estas condiciones se cumplen, procesamos la respuesta.

4. **Abrir la solicitud**: Usamos el método open() para configurar el tipo de solicitud (GET, POST, etc.) y la URL a la que se enviará.

5. **Enviar la solicitud**: Finalmente, enviamos la solicitud con el método send().

6. **Agregar un evento para ejecutar la solicitud**: Usamos un botón en la página que, al hacer clic, ejecutará la solicitud.

7. **Mostrar la respuesta en la página**: Si queremos que la respuesta se muestre en la página, podemos insertarla en un elemento de la página, como un div.

A continuación te muestro el código de ejemplo:


        // Crear un objeto XMLHttpRequest para la solicitud AJAX
        const xhr = new XMLHttpRequest();

        // Configurar el evento cuando el estado de la solicitud cambie
        xhr.onreadystatechange = function() {
            // Verificar si la solicitud se ha completado correctamente
            if (xhr.readyState === 4 && xhr.status === 200) {
                // Mostrar la respuesta en la consola
                console.log(xhr.responseText);
                
                // Insertar la respuesta en el div de la página
                document.getElementById("responseContainer").innerHTML = xhr.responseText;
            }
        };

        // Obtener el botón y agregar un evento para hacer la solicitud
        document.getElementById("loadDataBtn").addEventListener("click", function() {
            // Especificar el tipo de solicitud (GET) y la URL
            xhr.open("GET", "https://jsonplaceholder.typicode.com/posts/1", true);
            
            // Enviar la solicitud al servidor
            xhr.send();
        });
